static GameController& GameController::getInstance ( )
inlinestatic

The getInsatnce method of the gamecontroller.

This method makes sure there is only 1 instance of the gamecontroller at a time. This way, every time an external class uses a gamecontroller, it uses a gamecontroller with the same attributes as every other class.
